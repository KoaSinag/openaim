# Open AIM

**Building an open, secure and vendor-neutral standard for AI-readable web metadata.**

Open AIM is a community-driven project focused on developing an open specification that enables websites to publish structured, trustworthy and machine-readable metadata for software systems.

As the web continues to evolve, websites are increasingly consumed not only by people and search engines, but also by AI assistants, content management systems, accessibility tools, research platforms and other software. Open AIM seeks to provide a consistent and transparent way for websites to describe themselves and their content.

## Mission

To create an open, vendor-neutral specification that helps websites communicate structured, verifiable information to AI systems and software.

## Vision

A future where websites can publish clear, standardized metadata that improves understanding, interoperability and trust across the web.

## Guiding Principles

* Open and community-driven
* Vendor-neutral
* Transparent and trustworthy
* Security-conscious
* Extensible and future-proof
* Human-readable and machine-readable
* Free and open source

## What Open AIM Is

Open AIM is a standards initiative focused on defining a common format for website metadata.

The project explores how websites can communicate information such as:

* Website identity
* Ownership and contact information
* Content categories
* Supported languages
* Structured resources
* Content relationships
* Security and verification information
* Future AI and software integrations

## What Open AIM Is Not

Open AIM is not designed to:

* Manipulate search rankings
* Influence AI recommendations
* Replace existing web standards
* Provide advertising or promotional advantages

Open AIM exists to improve understanding between websites and the software that consumes them.

## Project Status

**Status:** Draft (Pre-RFC)

Open AIM is currently in the early planning and specification phase. The project is focused on defining its vision, principles and architecture before implementation begins.
