# Open AIM Timeline

This document records the major milestones in the history of the Open AIM Project.

The timeline is intended to preserve the project's evolution, important decisions, releases and community milestones.

---

## 2026

### 14 July 2026

#### The Idea

The initial concept for Open AIM was discussed.

The objective was to explore whether websites should have an open, standardized way of communicating structured, trustworthy information to AI systems and other software.

Early discussions focused on openness, transparency, interoperability and security as the project's guiding principles.

---

### 15 July 2026

#### Project Created

* Open AIM GitHub repository created.
* Project officially named **Open AIM**.
* The specification named **AIM (AI Manifest)**.
* Initial repository description published.

#### Foundational Decisions

* Open AIM will be community-driven.
* The specification will remain vendor-neutral.
* Development will begin with documentation and RFCs before implementation.
* GitHub will serve as the project's primary collaboration platform.
